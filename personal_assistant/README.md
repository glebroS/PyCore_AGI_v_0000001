# AGI v0.0000000001 (CLI Bot)

Цей проєкт є фінальним завданням курсу **Python Programming: Foundations and Best Practices**.

«AGI v0.0000000001» - це консольний застосунок для керування адресною книгою (контакти, номери телефонів, email, дні народження, фізичні адреси) та нотатками (текст, теги).

## Функціонал

### Контакти
- Додавання контактів з іменами, номерами телефонів, email, адресою та днем народження.
- Валідація номеру телефону (повинен містити 10 цифр).
- Валідація email за допомогою регулярних виразів.
- Валідація дня народження у форматі `DD.MM.YYYY`.
- Пошук контактів за іменем, телефоном або email.
- Виведення списку контактів, у яких день народження у найближчі N днів.
- Редагування імені, телефону, email, адреси контакту.
- Видалення контакту, телефону, email або адреси.

### Нотатки
- Збереження нотаток з текстовою інформацією.
- Підтримка багатослівних заголовків у лапках: `add-note "Мій заголовок" текст`.
- Перегляд окремої нотатки за заголовком.
- Пошук нотаток за заголовком або текстом.
- Редагування та видалення.
- Додавання та видалення тегів.
- Пошук нотаток за тегами.

### Збереження Даних
- Дані автоматично зберігаються у папці користувача (`~/personal_assistant_data/`).
- Перезапуск без втрати даних гарантовано!
- Всі дані зберігаються за допомогою бібліотеки `pickle`.

## Команди

### Управління контактами
| Команда | Опис |
|---|---|
| `add <ім'я> [телефон]` | Додати контакт |
| `change <ім'я> <старий> <новий>` | Змінити телефон |
| `remove-phone <ім'я> <телефон>` | Видалити телефон |
| `phone <ім'я>` | Показати телефони |
| `add-email <ім'я> <email>` | Додати email |
| `remove-email <ім'я> <email>` | Видалити email |
| `edit-email <ім'я> <старий> <новий>` | Змінити email |
| `add-address <ім'я> <адреса>` | Додати адресу |
| `remove-address <ім'я>` | Видалити адресу |
| `edit-name <старе> <нове>` | Змінити ім'я контакту |
| `delete <ім'я>` | Видалити контакт |
| `all` | Показати всі контакти |
| `search <запит>` | Пошук контактів |

### Управління днями народження
| Команда | Опис |
|---|---|
| `add-birthday <ім'я> <дата>` | Додати день народження (DD.MM.YYYY) |
| `show-birthday <ім'я>` | Показати день народження |
| `birthdays [дні]` | Найближчі дні народження (за замовчуванням 7 днів) |

### Управління нотатками
| Команда | Опис |
|---|---|
| `add-note "<заголовок>" <текст>` | Додати нотатку |
| `show-note <заголовок>` | Показати одну нотатку |
| `edit-note "<заголовок>" <текст>` | Змінити текст нотатки |
| `delete-note <заголовок>` | Видалити нотатку |
| `add-tag <заголовок> <тег>` | Додати тег |
| `remove-tag <заголовок> <тег>` | Видалити тег |
| `search-note <запит>` | Пошук нотаток за текстом |
| `search-tag <тег>` | Пошук нотаток за тегом |
| `all-notes` | Показати всі нотатки |

### Інші команди
| Команда | Опис |
|---|---|
| `hello` | Вітання від бота |
| `help` | Показати список команд |

| `speed-up-internet` | Прискорити інтернет 💳 |
| `close` / `exit` | Зберегти дані та вийти |
