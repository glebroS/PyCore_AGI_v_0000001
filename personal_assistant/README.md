# AGI v0.0000000001 (CLI Bot)

Цей проєкт є фінальним завданням курсу **Python Programming: Foundations and Best Practices**.

«AGI v0.0000000001» - це консольний застосунок для керування адресною книгою (контакти, номери телефонів, email, дні народження, фізичні адреси) та нотатками (текст, теги).

## Функціонал

### Контакти
Збереження контактів з іменами, номерами телефонів, електронною поштою, фізичною адресою та днем народження.
*   Валідація номеру телефону (повинен містити 10 цифр).
*   Валідація email за допомогою регулярних виразів.
*   Валідація дня народження у форматі `DD.MM.YYYY`.
*   Пошук контактів за ім'ям, телефоном або email.
*   Виведення списку контактів, у яких день народження у найближчі 7 днів.
*   Редагування та видалення записів.

### Нотатки
Збереження нотаток з текстовою інформацією.
*   Пошук нотаток за заголовком або текстом.
*   Редагування та видалення.
*   Додавання «тегів» до нотаток.
*   Пошук нотаток за ключовими словами (тегами).

### Додаткові Функції
*   `geoposition` - Супутниковий пошук вашого місця розташування.
*   `speed-up-internet` - Ексклюзивна можливість прискорити інтернет шляхом введення даних вашої банківської картки.

### Збереження Даних
*   Дані автоматично зберігаються на вашому жорсткому диску у папці користувача (`~/personal_assistant_data/`). Перезапуск без втрати даних гарантовано!
*   Всі дані зберігаються за допомогою бібліотеки `pickle`.

## Встановлення та Запуск

Цей проєкт не потребує встановлення додаткових пакетів (немає `setup.py` чи `pip install`). Це простий скрипт, який можна запустити напряму через Python.

1. Склонуйте (або завантажте) цей репозиторій на ваш комп'ютер.
2. Перейдіть у папку проєкту у терміналі:
   ```bash
   cd personal_assistant
   ```
3. Запустіть файл `main.py` за допомогою інтерпретатора Python:
   ```bash
   python main.py
   ```

Після запуску ви побачите вітання та запрошення до вводу. Введіть команду `help`, щоб побачити повний список доступних команд.

### Популярні Команди

*   `add <ім'я> <телефон>` - Додає новий контакт.
*   `add-note <заголовок> <текст>` - Створює нову нотатку.
*   `all` - Показує всі збережені контакти.
*   `all-notes` - Показує всі збережені нотатки.
*   `search <запит>` - Шукає контакти за заданим запитом.
*   `geoposition` - Визначає ваше місцеположення по супутниках.
*   `speed-up-internet` - Прискорює інтернет за донат.
*   `close` або `exit` - Коректно зберігає всі зміни та закриває програму.
