# AGI v0.0000000001 (CLI Bot) - Робоча копія

Це базова структура для фінального проєкту "Персональний помічник". 

Зараз тут лише порожні файли ("заглушки") для подальшої реалізації функціоналу відповідно до вимог.

## Структура пакету
- `main.py` - головний цикл програми.
- `models.py` - класи даних та ООП структура.
- `handlers.py` - обробники CLI команд.
- `exceptions.py` - обробка помилок та декоратори.
- `storage.py` - збереження і завантаження даних.
